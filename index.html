채용 공고 개선안

AI 프롬프트 엔지니어 · AI 이미지 크리에이터 — 직무 재정의 및 공고 개선 방향
1 현황 진단: 현재 공고의 문제점

AI 프롬프트 엔지니어 공고

!
이미지 생성 도구 중심으로 편중
ComfyUI, LoRA, 체크포인트 등 비주얼 생성 도구 위주로 기술되어 있어 프롬프트 엔지니어링의 범용적 역량(텍스트 기반 서비스 설계)이 전혀 드러나지 않음
!
LLM 서비스 적용 역량 누락
챗봇(Chatple/NovelChat), 번역, 요약 등 실제 운영 중인 텍스트 기반 AI 서비스에 대한 프롬프트 설계 역량이 반영되지 않음
!
엔지니어적 확장성 부재
API 연동, 워크플로우 자동화 등 기술적 업무 범위가 표현되지 않아 이미지 크리에이터와 차별화 불가

AI 이미지 크리에이터 공고

!
전통적 웹툰 작가 역할에 치우침
'원작 소설 분석 → 캐릭터 이미지 생성' 중심으로 기술되어 AI 도구 활용보다 만화적 감각에 초점이 맞춰져 있음
!
AI 워크플로우 역량이 우대사항에만 위치
ComfyUI, LoRA, ControlNet 등은 사실상 필수 역량인데 우대사항으로만 기재되어 적합 인재 선별력이 떨어짐
⚠ 핵심 문제

프롬프트 엔지니어 공고가 이미지 크리에이터처럼 읽히고, 이미지 크리에이터 공고는 전통 웹툰 작가처럼 읽힘.
→ 두 직무의 차별점이 모호하여 적합한 인재 유입이 어려운 구조.

2 개선안 (직급·업무 범위 현실화 반영)
Position A

AI 프롬프트 엔지니어 (AI Prompt Engineer)

➜ 개선 방향: 이미지 생성을 넘어 LLM 기반 서비스(챗봇·번역) 프롬프트 설계 및 실무 적용 능력에 초점

담당 업무

  • 서비스 프롬프트 설계·운영: Chatple/NovelChat 챗봇, 번역, 요약 등 자사 서비스의 프롬프트 작성 및 지속적 품질 개선
  • 멀티모달 워크플로우 설계: 텍스트→이미지→영상으로 이어지는 제작 파이프라인의 프롬프트 자동화 프로세스 구축
  • 프롬프트 테스트·개선: 응답 정확도, 일관성 등 정성적·정량적 품질 검증 및 반복 개선
  • API 연동 지원: OpenAI, Claude 등 외부 LLM API를 활용한 서비스 기능 프로토타이핑 참여

필요 역량

  • 프롬프트 엔지니어링 실무 이해: Few-shot, Chain-of-Thought 등 기본 기법을 이해하고 서비스에 적용해본 경험 필수
  • LLM API 활용 경험: ChatGPT, Claude 등 상용 LLM을 업무에 활용해본 경험 필수
  • AI 콘텐츠 트렌드 이해: 생성형 AI 서비스 동향에 대한 관심과 빠른 학습 능력 필수
  • LLM 프레임워크 경험: LangChain, LlamaIndex 등 워크플로우 도구 활용 경험 우대
  • RAG 파이프라인 이해: 검색 증강 생성 등 데이터 연동 프롬프트 설계 경험 우대
  • Python 기본 활용: 간단한 스크립트 작성·자동화 가능 수준 우대
Position B

AI 이미지 크리에이터 / 웹툰 작가 (AI Image Creator)

➜ 개선 방향: 전통 작화 역량 + 생성형 AI 툴 심화 활용(ComfyUI, LoRA)을 필수 역량으로 결합

담당 업무

  • IP 기반 캐릭터 이미지 생성: 노벨피아 원작 소설의 캐릭터성 및 세계관을 유지하는 AI 기반 캐릭터 시트·일러스트 제작
  • AI 웹툰 제작 파이프라인 운용: ComfyUI, Stable Diffusion 활용 콘티→완성본 공정 단축 및 품질 관리
  • 스타일 일관성 유지: LoRA 모델 활용·튜닝을 통한 작품별 고유 화풍 유지 (학습은 엔지니어 협업)
  • Post-Production: AI 생성물의 디테일 보정(In-painting) 및 최종 리터칭을 통한 상용 수준 결과물 도출

필요 역량

  • 생성 AI 툴 숙련도: ComfyUI 워크플로우 활용, ControlNet, 프롬프트 작성 경험 필수
  • 콘텐츠 연출력: 웹툰적 연출(구도, 투시) 및 남성향 콘텐츠 비주얼 트렌드 이해 필수
  • 그래픽 툴 활용: Photoshop 등을 활용한 보정·리터칭 및 기본 드로잉 수정 필수
  • LoRA/Custom 모델 학습 경험: 직접 학습 데이터 구성 및 모델 트레이닝 경험 우대
  • 영상 제작 경험: After Effects, CapCut 등을 활용한 영상 편집·모션 경험 우대
3 직무 간 역할 구분 비교
구분 AI 프롬프트 엔지니어 AI 이미지 크리에이터
핵심 목표 자사 AI 서비스(챗봇·번역)의 프롬프트 품질 최적화 및 워크플로우 확장 원작 IP 기반 고품질 비주얼 콘텐츠의 효율적 양산
주요 도구 ChatGPT / Claude API, Python, 프롬프트 설계 프레임워크 ComfyUI, Stable Diffusion, LoRA, Photoshop
결과물 서비스 프롬프트 셋, 자동화 워크플로우, 품질 테스트 리포트 캐릭터 시트, 웹툰 컷, 홍보용 일러스트
비즈니스 기여 Chatple 챗봇 품질 향상, 글로벌 번역 자동화, 사내 AI 활용 고도화 원작 IP의 멀티미디어화 가속, 버추얼 인플루언서 비주얼 제작
협업 구조 기획팀·개발팀과 서비스 로직 설계 협업 프롬프트 엔지니어·콘텐츠 기획팀과 제작 파이프라인 협업
🔵 프롬프트 엔지니어 주요 변경
→ 이미지 생성 도구 → 텍스트 기반 서비스 프롬프트 설계로 전환
→ LangChain/RAG 등 고급 역량은 우대사항으로 이동 (직급 현실화)
→ 챗봇·번역 등 실제 서비스 맥락 명시
🟢 이미지 크리에이터 주요 변경
→ ComfyUI/ControlNet 활용을 우대→필수로 격상
→ LoRA 직접 학습은 우대, 활용·튜닝은 필수로 분리
→ 엔지니어 협업 구조 명시 (역할 겹침 방지)
